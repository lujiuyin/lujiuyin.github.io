hexo一些命令

发表于 2017-12-23

创建一篇文章

hexo n “custom title” == hexo new “custom”

也可以指定自己写好的scaffolds中的layout快捷生成模版(post.md、draft.md等),自己可以定义

hexo n [layout] “custom title” == hexo new [layout] “custom”

新增、修改的文件生效

hexo g == hexo generate

部署

hexo d == hexo deploy

启动服务预览

hexo s == hexo server

GCD使用总结

发表于 2017-12-18 | 分类于 iOS开发学习

iOS开发中gcd使用的个人理解

首先当然是一些术语的介绍:

同步&异步:同步即在当前线程执行任务,异步即新开一个线程执行任务.

串行&并行:串行即任务有顺序的一个执行完了之后再去执行下一个,并行即多个任务一起执行.

多线程中队列有: 串行队列、并行队列、全局队列、主队列

队列是包含一系列任务的队列,队列运行在线程中.
