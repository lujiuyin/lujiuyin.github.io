UITableView偏移设置

发表于 2017-11-03 | 分类于 iOS开发学习

UITableView内容偏移

Frame

定义了UIScrollView滚动视图的边框。

frame = (0,0,320,568);

contentSize

UIScrollView可以滚动的区域,contentSize.height就是内容的总高度。

contentSize = (320,800);

contentOffset

内容的偏移量，其中contentOffset.y就是内容的顶部与frame顶部的差值。contentOffset.x就是内容的左边与frame左边的差值。

内容向上偏移为正，向下偏移为负。

contentInset

内边距，在内容周围增加的边距。contentInset的类型是UIEdgeInsets，默认为UIEdgeZero。

总结一下

tableView的内容(即contentSize):

1> cell

2> tableHeaderView\tableFooterView

3> sectionHeader\sectionFooter

contentSize.height = 内容的高度

contentOffset.y = 内容的顶部 和 frame顶部 的差值

contentInset = 在内容周围增加的间距（粘着内容）
