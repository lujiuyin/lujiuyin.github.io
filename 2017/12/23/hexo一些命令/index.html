hexo一些命令

发表于 2017-12-23

创建一篇文章

hexo n “custom title” == hexo new “custom”

也可以指定自己写好的scaffolds中的layout快捷生成模版(post.md、draft.md等),自己可以定义

hexo n [layout] “custom title” == hexo new [layout] “custom”

新增、修改的文件生效

hexo g == hexo generate

部署

hexo d == hexo deploy

启动服务预览

hexo s == hexo server
